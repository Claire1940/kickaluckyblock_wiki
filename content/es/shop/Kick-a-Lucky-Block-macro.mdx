<Section icon={Target} title="Configurando tu Macro Kick a Lucky Block">

Para usar eficazmente un macro Kick a Lucky Block, necesitas una herramienta fiable. Aunque muchos jugadores usan auto-clickers básicos, herramientas como TinyTask o grabadoras de macros especializadas permiten la grabación de movimientos. Esto es necesario porque el juego requiere que corras de regreso a tu parcela para "depositar" el Brainrot ganado con una patada. Sin el viaje de regreso, el tsunami eventualmente te atrapará y perderás la recompensa.

<Steps>
  <Step num="1" title="Posicionando a tu Personaje">
    Ponte en el centro exacto de tu parcela, mirando el Lucky Block. Asegúrate de que tu cámara esté bloqueada o configurada en un ángulo consistente para evitar que el macro se desvíe con el tiempo.
  </Step>
  <Step num="2" title="Grabando la Fase de Entrenamiento">
    Si tu objetivo es Kick Power, grábate haciendo clic con tu mejor pesa equipada. La mayoría de las pesas, como la Barra de Hueso o la Placa de Cobre, requieren clics constantes para generar poder.
  </H2>
  <Step num="3" title="Grabando el Ciclo de Farmeo">
    Graba la siguiente secuencia: Patea el bloque -> Espera el aterrizaje -> Corre de regreso a la parcela inmediatamente -> Espera 2 segundos para que se registre el Brainrot -> Repite.
  </Step>
  <Step num="4" title="Probando la Sincronización del Tsunami">
    Reproduce tu grabación. Si el tsunami te atrapa durante la reproducción, necesitas aumentar tu Velocidad de Carrera o acortar el tiempo de espera en tu macro.
  </Step>
</Steps>

| Herramientas Esenciales de Macro | Función Principal | Mejor Para | Compatibilidad |
| | | | |
| OP AutoClicker | Clics rápidos | Entrenamiento de Pesas | Windows/Mac |
| TinyTask | Grabación de movimiento | Ciclos completos de Farmeo | Windows |
| MacroRecorder | Lógica compleja | Automatización de Renacimiento | Windows/Mac |

<Callout type="warning" title="Conciencia Anti-Trampas">
