## Loop Farming Terbaik Sebelum Return Celestial Penuh

Sampai movement kamu bisa menangani tekanan wave Celestial, jalankan loop farm terkontrol yang memberi makan tier speed berikutnya. Ini lebih efisien daripada berjudi pada push dalam yang gagal berulang.

### Loop A: Loop Pertumbuhan Aman (Kebanyakan Pemain)

- Tumpuk power dengan OG barbell dalam window AFK singkat  
- Tendang ke lane yang bisa kamu return secara konsisten (seringnya Hacked atau awal OG)  
- Bawa pulang unit bernilai menengah-tinggi setiap run  
- Levelkan hanya penghasil teratasmu; hindari menyebar upgrade terlalu tipis  
- Investasikan ulang ke run speed di setiap checkpoint keterjangkauan

### Loop B: Loop Push Agresif (Lanjutan)

- Naikkan ke checkpoint power tinggi (misalnya 300M+)  
- Coba run dalam dekat threshold Celestial  
- Jika return gagal dua kali berturut-turut, turunkan lane dan bangun ulang currency  
- Beli upgrade speed, coba lagi push dalam dengan timing kick sempurna

| Tipe Loop | Risiko | Stabilitas Income | Cocok Untuk |
|---|---|---|---|
| **Pertumbuhan Aman** | Rendah | **Tinggi** | Pemilik OG baru |
| **Push Agresif** | Sedang-Tinggi | Sedang | Pemain berpengalaman |
| **Hybrid (2 aman, 1 push)** | Sedang | **Tinggi-Sedang** | Paling efisien secara keseluruhan |

Pola hybrid biasanya paling bagus dengan **Kick a Lucky Block og barbell** karena melindungi ekonomi kamu sambil tetap memberi percobaan progres.

## Kesalahan yang Memperlambat Pemain OG Barbell

Bahkan dengan weight terbaik sekalipun, progres bisa terasa macet jika alokasi kamu meleset. Ini kesalahan terbesar yang perlu langsung diperbaiki:

1. **Terlalu over-invest pada satu dimensi**  
   Build yang hanya power akan mentok di dinding return. Build yang hanya speed kehilangan value jarak.

2. **Mengabaikan timing kualitas kick**  
   Percobaan “Perfect” penting karena mengubah kedalaman lane dan peluang drop.

3. **Memaksa Celestial terlalu dini**  
   Jika kamu menyentuh Celestial tapi ekstraksi yang didapat masih tier OG, farm OG/Hacked dengan aman sampai speed naik.

4. **Menaikkan level terlalu banyak unit sekaligus**  
   Fokuskan resource ke unit ROI tertinggi untuk pertumbuhan currency yang lebih baik.

5. **Tanpa struktur siklus**  
   Main acak lebih lambat daripada loop tetap (AFK squat → beli speed → farm aman → tes push).

| Kesalahan | Gejala | Perbaikan Cepat |
|---|---|---|
| **Belanja hanya power** | Jangkauan jauh, return gagal | Beli speed dalam 1–2 pembelian berikutnya |
| **Belanja hanya speed** | Return stabil, kedalaman lane lemah | Jalankan siklus AFK squat dengan OG barbell |
| **Tidak fokus ROI** | Pertumbuhan currency terasa datar | Levelkan 2–3 penghasil teratas dulu |
| **Spam push** | Run Celestial gagal berulang | Gunakan 2 loop aman per 1 percobaan push |

## FAQ

**Q: Apakah Kick a Lucky Block og barbell layak dibeli dulu dibanding upgrade kecil?**

A: Di sebagian besar jalur progres, ya. OG barbell memberi boost besar power per aksi yang meningkatkan pipeline squat-ke-jarak kamu. Tinggal pasangkan dengan upgrade speed segera setelahnya agar return tetap menguntungkan.

**Q: Kenapa saya mencapai Celestial tapi hasilnya masih kebanyakan tier OG?**

A: Kemungkinan kamu mendarat di tepi awal Celestial atau gagal ekstraksi karena tekanan wave. Tingkatkan run speed dan perbaiki timing kick agar kamu bisa masuk lebih dalam dan return dengan sukses.

**Q: Seberapa sering saya harus AFK squat dengan OG barbell?**

A: Window pendek yang bisa diulang (sekitar 10–20 menit) biasanya efisien. Setelah tiap siklus, uji kedalaman lane, lalu putuskan apakah akan farm aman atau mencoba push lebih dalam.

**Q: Apa fallback terbaik jika run Celestial terus gagal dengan Kick a Lucky Block og barbell?**

A: Gunakan loop hybrid: dua run farm aman Hacked/OG, lalu satu percobaan push Celestial. Ini menjaga income tetap jalan sambil tetap menaikkan ceiling progres kamu dari waktu ke waktu.
