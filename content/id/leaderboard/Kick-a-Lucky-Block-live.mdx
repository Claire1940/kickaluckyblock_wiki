Saat unit base yang jauh lebih kuat ditambahkan (misalnya class yang scale-nya jauh melampaui setup sebelumnya), berkomitmenlah mengganti semua slot alih-alih mencampur lama dan baru. Base campuran terlihat oke, tapi memperlambat compounding-mu.

## Prioritas Base dan Upgrade Terbaik untuk Uang Cepat

Kebanyakan progresi macet terjadi di layar base, bukan di animasi kick. Base-mu adalah mesin multiplier.

### Aturan inti
Jika class unit base baru jauh lebih kuat daripada setmu saat ini, lakukan **konversi penuh**. Konversi parsial menunda pertumbuhan income pasif dan membuat tiap menit jadi kurang bernilai.

### Urutan upgrade (direkomendasikan)
1. **Unit carry rare utama** (unit per detik tertinggi)  
2. **Slot class base teratas** (terutama jika semua slot class-nya sama)  
3. **Unit rare sekunder** hanya jika benar-benar melampaui rata-rata base  
4. **Upgrade utility** (speed/QoL) setelah ekonomi stabil

| Target Upgrade | Prioritas | Alasan |
|---|---|---|
| **Carry rare teratas** | **Sangat Tinggi** | ROI langsung terbaik |
| **Class base seragam** | **Tinggi** | Scaling pasif lebih mulus |
| **Rare terbaik kedua** | **Sedang** | Bagus saat carry utama mulai mahal |
| **Boost kosmetik/impact rendah** | **Rendah** | Dampak progresi minimal |

> **⚠️ Peringatan:** Jangan menghabiskan semua mata uang ke satu tier upgrade jika itu menghambat power checkpoint berikutnya. Keseimbangan itu penting saat mendekati zona baru.
